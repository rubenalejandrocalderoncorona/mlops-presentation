# MLOps Interactive Presentation

An interactive presentation built with React, Vite, TypeScript, Tailwind CSS, and Framer Motion. All content is in Spanish.

## Requirements

- Node.js 18+
- npm 9+

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) in your browser.

## Navigation

| Key | Action |
|-----|--------|
| `→` / `↓` | Next slide |
| `←` / `↑` | Previous slide |
| `Space` | Advance animation step (on animated slides) |
| `B` | Go back one animation step |

## Build for Production

```bash
npm run build
npm run preview
```

The `dist/` folder contains the static site ready to serve.
